## Using Genre Sort
(windows)
* Open Terminal
    * cd {location of the downloaded .tar file}
    * docker load -i genre-sort.tar
    * docker run -p 5000:5000 genre-sort
* Open Web Browser
    * Go to 'localhost:5000
